FreeFlume

Fast native desktop YT client. Install into your user account (no root) from this self-hosted Flatpak repo — updates arrive through flatpak update / your software center:

flatpak install --user --from https://velkadyneslop.github.io/freeflume-linux-qt6/com.velkadyne.FreeFlume.flatpakref

Then run flatpak run com.velkadyne.FreeFlume. Update later with:

flatpak update --user com.velkadyne.FreeFlume
